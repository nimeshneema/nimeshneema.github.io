Apple Developer Tools, Technologies, Frameworks and Programming Languages

Apple Operating System Platforms:

  • macOS: macOS, originally known as Mac OS X, is an operating system developed for Mac lineup of desktop and laptop computers.
  • iOS: iOS is a mobile operating system developed for use exclusively on iPhone.
  • iPadOS: iPadOS is a mobile operating system developed for iPad line of tablet computers.
  • watchOS: watchOS is the operating system of the Apple Watch. It is based on iOS, and has many similar features.
  • tvOS: tvOS is an operating system developed for the Apple TV, a digital media player.
  • visionOS: visionOS is a mixed reality operating system for Apple's first spatial computing device Apple Vision Pro.

Developer Tools & Technologies:

  • Xcode: Apple's integrated development environment for macOS, used to develop software for iOS, iPadOS, macOS, watchOS, tvOS, and visionOS.
  • Xcode Cloud: Xcode Cloud is a continuous integration and delivery service built into Xcode and designed expressly for Apple developers.
  • Instruments: Instruments is a performance-analysis and testing tool for macOS, used to analyse iOS, iPadOS, macOS, watchOS, tvOS, and visionOS apps.
  • TestFlight: TestFlight is an online service for over-the-air installation and testing of apps for iOS, iPadOS, macOS, watchOS, tvOS, and visionOS.
  • Swift Package Manager (SwiftPM): Swift Package Manager is a tool for managing distribution of source code, aimed at making it easy to share your code and reuse others' code
  • Swift Playgrounds app: Swift Playgrounds is a development environment for Swift programming language, available as an iPadOS and macOS app.
  • Mac Catalyst: Mac Catalyst lets developers easily create a native Mac app from iPad app codebase.
  • App Store Connect: App Store Connect is an Apple service used for distribution of software, media and consumables via Apple's online digital stores.
  • Transporter: Transporter is a macOS app used for delivering software, media and consumables to Apple.
  • Terminal: Terminal is the terminal emulator bundled with macOS.

Frameworks:

  • Cocoa Framework: Cocoa is Apple's native object-oriented application programming interface (API) for developing apps for macOS.
  • Cocoa Touch Framework: Cocoa Touch is the application development environment for building software programs to run on iOS, iPadOS, watchOS and tvOS.
  • Foundation Framework: Foundation framework provides a base layer of functionality for apps and frameworks, including data storage and persistence, text processing, date and time calculations, sorting and filtering, and networking.
  • AppKit Framework: AppKit framework contains the objects required to build the user interface for a macOS app.
  • UIKit Framework: UIKit framework provides a variety of features for building apps, including components required to construct the core infrastructure of iOS, iPadOS, watchOS and tvOS.
  • SwiftUI: SwiftUI is a declarative framework used for creating user interface and behavior apps on iOS, iPadOS, macOS, watchOS, tvOS, and visionOS.

Programming Languages:

  • C Programming Language: C is a general-purpose computer programming language that acts as a foundation for many modern high level computer programming languages.
  • Objective-C Programming Language: Objective-C is a high-level general-purpose, object-oriented programming language, used for creating apps for Apple operating system platforms.
  • Swift Programming Language: Swift is a high-level general-purpose, multi-paradigm, compiled programming language, commonly used for developing apps for Apple operating system platforms.
  • AppleScript: AppleScript is a scripting language that facilitates automated control over scriptable Mac applications.
